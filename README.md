
# 📱 Customer Sentiment Analysis — iPhone 15 128GB

Sentiment analysis of 300+ Flipkart customer reviews using Selenium, BeautifulSoup, Pandas, TextBlob, and Matplotlib.

## Tools Used
- Selenium & BeautifulSoup — Web scraping
- Pandas — Data cleaning
- TextBlob — Sentiment analysis
- Matplotlib/Seaborn/WordCloud — Visualizations
